Many websites designed to engage Canadians in the political process need a way to let people find out who their MP is. This is usually done with a postal code.

Political parties do not need to pay to have this on their websites. Advocacy organizations however must pay $2,500 the first year, $500/yr thereafter.

This is a free alternative, based on data from Elections.ca.

One legal opinion we received indicated this was legal so long as we did not make an unreasonable number of requests to their servers. For this reason, we only forward requests when they are made to us. We also make a copy of the result to avoid subsequent lookups for the same postal code.

This service is powered by open source software. To use the API, refer to the README

Others have tried to lobby for access to this data for years. Notably, Russell McOrmond wrote about has been documenting his efforts since 2005: Postal codes by federal ridings (2003 representation order) file (PCFRF) tools.
